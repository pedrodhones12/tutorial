Como Manusear Seu Novo Site

Introdução ao HTML

HTML (HyperText Markup Language) é a estrutura básica de um site. Aqui está um exemplo simples de um documento HTML:

<!DOCTYPE html>
<html>
<head>
<title>Título da Página</title>
</head>
<body>
<h1>Olá, mundo!</h1>
</body>
</html>

Introdução ao CSS

CSS (Cascading Style Sheets) é usado para estilizar o seu site. Aqui está um exemplo de como incluir CSS no seu HTML:

<style>
body {
  background-color: #f4f4f4;
  font-family: Arial, sans-serif;
}
h1 {
  color: #333;
}
</style>

Introdução ao JavaScript

JavaScript é usado para adicionar interatividade ao seu site. Aqui está um exemplo simples de JavaScript:

<script>
function showMessage() {
  alert('Olá, mundo!');
}
</script>
<button onclick="showMessage()">Clique em mim</button>
